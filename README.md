# 📦 Logistics & Route Optimization using Algorithms

This repository showcases a set of Python implementations designed to model and solve common delivery and logistics problems using fundamental algorithmic techniques.

## 📁 Contents

* `model.py` – Models delivery locations, parcels, and vehicle constraints
* `tsp_rec.py` – Recursive solution for Travelling Salesman Problem (TSP)
* `greedy.py` – Greedy approach for parcel selection (value/weight ratio)
* `knapsack_dp.py` – Dynamic Programming solution for profit maximization
* `dijkstra.py` – Shortest path calculation from warehouse
* `mst.py` – Minimum Spanning Tree using Prim’s Algorithm
* `tsp.py` – Brute-force TSP using permutations

## 🔍 Key Highlights

* Applies algorithms to a **real-world delivery scenario**
* Covers **optimization, graph traversal, and decision-making techniques**
* Simple, readable, and easy-to-understand code
